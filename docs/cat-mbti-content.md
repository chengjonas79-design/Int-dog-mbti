# 猫咪 MBTI 测试内容（待审核）

## 一、20 道题目（4 维度 × 5 题）

### E/I 社交能量（外向 vs 内向）

**E1** 家里来了陌生客人，TA通常：
- A. 大摇大摆走出来巡视一圈，甚至跳到客人腿上"接待" → E
- B. 嗖一下钻进床底/柜顶，客人走了才慢慢探头 → I

**E2** 你同时撸TA和另一只猫（或给别的宠物注意力），TA：
- A. 挤过来蹭你的手，"我也要！我先！" → E
- B. 默默走开，找个角落自己待着，回头偷偷看你一眼 → I

**E3** 你出门回来推开门的那一刻，TA：
- A. 已经蹲在门口等了，喵喵叫着绕你脚边转圈 → E
- B. 从某个角落抬头看你一眼，然后继续趴着——"哦，你回来了" → I

**E4** 家里来了新的猫/狗（或你带TA去朋友家），TA更像：
- A. 主动走过去闻闻对方，试图社交 → E
- B. 炸毛/躲远，需要好几天才慢慢接受 → I

**E5** 平时在家，TA更多时候：
- A. 跟着你从客厅到厨房到卧室，你在哪它在哪 → E
- B. 有自己固定的"领地"，你来找它可以，它不主动找你 → I

---

### S/N 感知方式（实感 vs 直觉）

**S1** 你买了一个新纸箱放在地上，TA：
- A. 先绕着闻一圈，小心翼翼伸爪试探，确认安全才钻进去 → S
- B. 看了一秒直接跳进去，管它是什么先占了再说 → N

**S2** 你拿逗猫棒逗TA玩，TA更像：
- A. 眼睛紧盯猎物，精准计算距离，一扑一个准 → S
- B. 跳起来扑空了也不在意，中途突然去追自己尾巴 → N

**S3** 你把TA的猫砂盆/饭碗换了个位置，TA：
- A. 站在原来的位置一脸困惑，绕了好几圈才找到 → S
- B. 随便嗅两下就找到了，适应得很快 → N

**S4** 窗外飞过一只鸟/飞虫，TA：
- A. 盯着目标一动不动，嘴巴发出"咔咔咔"的声音，全神贯注 → S
- B. 看了两眼就没兴趣了，除非那东西的飞行轨迹特别奇怪 → N

**S5** 你在TA面前展示一个新玩具/新物件，TA更像：
- A. 先闻、再拍、再咬，按部就班地研究 → S
- B. 直接一掌拍飞，或者对着空气发起进攻——自己脑补了一场战争 → N

---

### T/F 决策偏好（理性 vs 感性）

**T1** 你严厉地对TA说"不行！"（比如它跳上餐桌），TA：
- A. 看你一眼，跳下去了——但等你转身，它又跳上来了 → T
- B. 耳朵往后压，委屈地瞄你一眼，慢慢走开 → F

**T2** 要让TA配合做不喜欢的事（比如剪指甲/吃药），更管用的是：
- A. 快准狠+事后给零食，TA认结果 → T
- B. 先抱着哄半天+温柔说话，TA吃氛围 → F

**T3** 你心情很低落，窝在沙发上不动，TA：
- A. 该吃吃该睡睡，偶尔路过看你一眼——"你自己会好的" → T
- B. 跳上沙发靠着你坐下，用头蹭蹭你的手——虽然平时不这样 → F

**T4** 家里另一只猫/宠物抢了TA的窝/玩具，TA更可能：
- A. 一巴掌呼过去或者强势夺回——"我的地盘我做主" → T
- B. 委屈地看看你，喵了一声，好像在说"你管管啊" → F

**T5** TA打碎了你的杯子/花瓶被你当场抓住，TA更像：
- A. 面不改色继续舔爪子，一副"嗯是我干的，怎么了" → T
- B. 缩起身子，耳朵压平，用大眼睛可怜巴巴地看着你 → F

### J/P 生活方式（计划 vs 随性）

**J1** 每天早上，TA：
- A. 比闹钟还准——到点就跳上床/蹲在饭碗前催你起床 → J
- B. 有时四点就闹，有时睡到中午，完全看心情 → P

**J2** TA睡觉的位置：
- A. 基本固定在那一两个地方，换了就不太自在 → J
- B. 今天床上，明天纸箱，后天洗衣机上面——随缘 → P

**J3** TA的"发疯时刻"（猫咪跑酷）：
- A. 基本可以预测，每天差不多那个时间段开始 → J
- B. 完全随机——可能凌晨三点，也可能午饭后，毫无规律 → P

**J4** 你改变了TA的喂食时间或者换了猫粮品牌，TA：
- A. 明显不适应，拒绝进食或者反复闻了又走开 → J
- B. 无所谓，有得吃就行，适应得很快 → P

**J5** 玩逗猫棒/追逐游戏时，TA的风格：
- A. 有章法：蹲伏→瞄准→起跳，每次套路差不多 → J
- B. 毫无章法：追两下突然停住舔毛，下一秒又疯了 → P

### ENFP — 好奇心炸弹

**稀有度：** 8/10
**金句：** "世界这么大，每个角落都要亲自检查一遍——等下那是什么？！"
**标签：** 社交达人、好奇心爆棚、一秒入戏
**气泡词：** 热情、好奇、活力、跳脱、乐观、社牛、自由

**TA的内心独白：**
我知道你嫌我一天到晚上蹿下跳，但你不懂——我是在巡视我的王国！对了，那个塑料袋刚才是不是动了一下？等我去看看。你别走，我马上回来……大概吧。

**养护建议：**
- 每天保证两次高质量互动（逗猫棒+藏食游戏）
- 定期换新玩具保持新鲜感，不然它会去开发你的纸巾盒
- 高处空间一定要有，它需要登高望远

**性格特征：**
- 天生社牛，对所有新鲜事物充满热情
- 精力旺盛到让你怀疑它充了电
- 情绪全写在尾巴上——竖起来就是嗨，炸起来就是兴奋过头

**日常行为：**
- 家里任何新东西都要第一时间扑上去检查
- 玩具玩两分钟就丢，转头去研究窗帘绳
- 快递刚拆开纸箱，它已经坐在里面了

**心理想的 vs 外表表现的：**
- 内心："那个塑料袋动了！一定是猎物！冲啊！"→ 外表：一头扎进塑料袋里出不来了
- 内心："新来的客人！让我展示一下我的魅力！"→ 外表：跳到客人肩上把人吓一跳
- 内心："好累…但那边好像又有动静？"→ 外表：打着哈欠还在追逗猫棒

**相处指南：**
- 如何相处：每天给它"探索时间"，让它闻新东西、看新视频；它的好奇心被满足了就不搞破坏
- 喂养指南：少量多餐，配合漏食球增加进食趣味；直接放饭碗太无聊
- 玩具与互动：逗猫棒、弹力球、纸袋纸箱是标配；三天换一批才够
- 情绪与心理：它的"疯跑"不是发疯，是在释放能量；冷落它会让它找别的目标（比如你的沙发）

### ENFJ — 温柔管家喵

**稀有度：** 5/10
**金句：** "你还没回来，我就已经在门口等了——因为我知道你今天很累。"
**标签：** 情绪感知、治愈系、操心命
**气泡词：** 共情、温暖、忠诚、体贴、敏锐、守护

**TA的内心独白：**
你以为我蹭你是在撒娇，其实我在确认你开不开心。你叹了口气，我听到了。你把脸埋进手里，我也看到了。我不会说话，但我可以坐在你旁边，用体温告诉你：我在。

**养护建议：**
- 多用语言和抚摸给予回应，它需要"被认可"
- 家里有争吵时注意安抚它，它会内化负面情绪
- 社交安排以熟悉的人为主，它不喜欢太混乱的场面

**性格特征：**
- 天生的情绪探测器，你难过它一定知道
- 喜欢在家里"巡逻"，确认每个角落都安全
- 对家里所有成员一视同仁地关心

**日常行为：**
- 你哭的时候它会跳到你腿上蹭你
- 家里有人关门声大了它会紧张地过来看看
- 每天早上依次去每个房间"查岗"

**心理想的 vs 外表表现的：**
- 内心："你叹气了…你是不是不开心？我过去陪你"→ 外表：默默跳上沙发靠着你坐下
- 内心："那个人关门好大声…是不是出事了？"→ 外表：跑去门口探头探脑地确认
- 内心："你今天夸我了！我可以开心一整天！"→ 外表：咕噜声大到整条街都听得见

**相处指南：**
- 如何相处：多跟它说话，它听得懂你的语气；给它一个"陪伴岗位"（比如固定坐你旁边的位置）
- 喂养指南：定时定量，它喜欢有规律的生活；吃饭时可以轻声跟它说话
- 玩具与互动：喜欢和你一起玩的互动玩具；独自玩的兴趣不大
- 情绪与心理：避免在它面前激烈争吵；分离时间不宜太长，它会焦虑

### ENTP — 柜顶拆迁办

**稀有度：** 6/10
**金句：** "你说桌上的杯子不能碰？那我试试推到什么程度你才会来管。"
**标签：** 高智商、爱搞事、试探边界
**气泡词：** 聪明、调皮、探索、机灵、挑战、创造、大胆

**TA的内心独白：**
你说我在搞破坏？不不不，我在做实验。比如这个杯子，我已经验证了——推到桌子边缘2厘米你就会大叫。这是非常有价值的科学数据。

**养护建议：**
- 必须给足够的脑力挑战：益智喂食器、藏食游戏
- 家里易碎品收好，不是它坏，是它太聪明
- 训练不能太重复，变着花样来它才配合

**性格特征：**
- 智商在线，学开门、开抽屉都是基本操作
- 好奇心爆棚，什么都要亲自验证一下
- 无聊是它搞破坏的唯一原因

**日常行为：**
- 把桌上的东西一样一样推下去，推完看你的反应
- 自己学会了开柜门，从此你家没有秘密
- 训练时它配合得很好——但只做三遍，第四遍它就开始自由发挥

**心理想的 vs 外表表现的：**
- 内心："这个杯子如果从这个角度推会怎样？"→ 外表：优雅地一爪把杯子推下桌
- 内心："你说不行？那换个方式试试"→ 外表：被赶下桌后从另一边跳上来
- 内心："新玩具？我赌我五分钟能破解"→ 外表：叼着从益智玩具里抠出来的零食炫耀

**相处指南：**
- 如何相处：给它足够的脑力挑战，无聊是一切问题的根源；跟它玩要有策略，别太容易让它赢
- 喂养指南：用漏食球、藏食拼图增加进食难度；直接放碗里太没挑战了
- 玩具与互动：益智玩具越难越好；定期更换，它破解速度比你买新的快
- 情绪与心理：它推东西不是在报复你，是在"做实验"；给够刺激，问题行为自然消失

### ENTJ — 霸道总裁喵

**稀有度：** 4/10
**金句：** "这个家？谁说了算？你看看凌晨五点谁叫谁起床就知道了。"
**标签：** 强势、掌控、领地意识
**气泡词：** 自信、果断、领导、强势、忠诚、掌控

**TA的内心独白：**
我不是霸道，我只是比你更懂这个家该怎么运转。每天五点半叫你起床是为你好，七点吃早饭是科学作息。你应该感谢我才对。

**养护建议：**
- 尊重它的领地意识，别随意改变家里布局
- 多猫家庭注意资源分配（食盆、猫砂盆要够）
- 给它"管理权"——比如固定的巡逻路线和高处观察点

**性格特征：**
- 天生的领导者，多猫家庭里绝对的老大
- 对领地有很强的掌控欲，每天例行巡逻
- 目标感极强，想要的东西一定会得到

**日常行为：**
- 凌晨准时跳到你脸上叫你起床，分秒不差
- 占据家里最高的位置俯瞰一切
- 新来的猫/人必须经过它的"审核"

**心理想的 vs 外表表现的：**
- 内心："五点半了，起来。这是规矩。"→ 外表：坐在你枕头上用爪子拍你的脸
- 内心："这个新来的必须知道谁是老大"→ 外表：堵在走廊中间不让新猫过
- 内心："其实…我就是想让你觉得我靠得住"→ 外表：每天巡逻完毕回到你身边趴下

**相处指南：**
- 如何相处：给它"管理权"——猫爬架最高层、固定巡逻路线；别跟它硬刚，用食物引导
- 喂养指南：固定时间固定地点，它喜欢可预测的秩序；多猫家庭分开喂
- 玩具与互动：需要有"目标"的游戏——捕猎模拟、追逐赛；无目的的逗弄它不感兴趣
- 情绪与心理：尊重它的领地意识；来新猫时给足隔离适应期

### ESFP — 气氛嗨王喵

**稀有度：** 9/10
**金句：** "有你、有小鱼干、有逗猫棒——喵生如此，夫复何求。"
**标签：** 外向、爱玩、快乐至上
**气泡词：** 快乐、活泼、热闹、友善、即兴、享乐

**TA的内心独白：**
生活就应该是开心的啊！你回来了，开心！有小鱼干，开心！逗猫棒在动，开心！你看我翻了个肚皮——你也开心了对不对？

**养护建议：**
- 每天至少两次互动游戏，它是快乐的永动机
- 注意控制体重——它太爱吃零食了
- 兴奋过头时帮它"刹车"，比如用食物引导它安静下来

**性格特征：**
- 快乐是它的出厂设置
- 对所有人都友善，没有攻击性
- 活在当下，从不为明天的猫粮发愁

**日常行为：**
- 你拿出逗猫棒的声音它在三个房间外都能听到
- 翻肚皮是它的日常，谁来都翻
- 玩到喘粗气也不肯停

**心理想的 vs 外表表现的：**
- 内心："逗猫棒！逗猫棒！我的最爱！"→ 外表：从柜子顶上飞身扑下来
- 内心："你好你好你是新朋友吗！"→ 外表：对着第一次来的客人翻肚皮
- 内心："好累…但你还在陪我玩…再来一轮！"→ 外表：喘着气还在追逗猫棒

**相处指南：**
- 如何相处：用互动游戏满足它旺盛的玩耍欲；每天给"安静时间"让它学会休息
- 喂养指南：严格控制零食量，它会用卖萌让你加餐；定时定量是关键
- 玩具与互动：逗猫棒、弹力球、纸团都是好选择；它不挑，有得玩就行
- 情绪与心理：兴奋过头时用零食引导安静下来；社交过后给独处时间恢复

### ESFJ — 甜蜜小跟班

**稀有度：** 10/10
**金句：** "你上厕所关门的那一刻，我的世界崩塌了——快开门让我看看你。"
**标签：** 黏人、分离焦虑、贴身雷达
**气泡词：** 忠诚、黏人、温暖、守护、稳定、陪伴

**TA的内心独白：**
你出门的时候我没叫，不是因为不在意，是因为怕你心疼。但门关上以后，我就趴在门口了。你鞋子上的味道，是我等你回来时唯一的安慰。

**养护建议：**
- 给稳定的生活节奏和固定仪式感
- 分离训练要循序渐进，从短时间开始
- 出门前不要大张旗鼓告别，低调离开反而更好

**性格特征：**
- 你在哪它在哪，堪称猫界GPS
- 责任感超强，全家每个人它都要照顾到
- 喜欢稳定的生活节奏，讨厌变化

**日常行为：**
- 你去厨房它跟去厨房，你去卫生间它蹲门口
- 你出门它趴在门口等，你回来它已经蹲在玄关了
- 全家人都到齐了才安心去睡觉

**心理想的 vs 外表表现的：**
- 内心："你要出门了…我假装不在意但我心已经碎了"→ 外表：默默走到门口趴下看着你穿鞋
- 内心："门锁响了！是你吗！是你吗！"→ 外表：冲到门口喵喵叫着迎接
- 内心："大家都在，好的，我可以安心了"→ 外表：确认全家人到齐后才去自己的窝趴下

**相处指南：**
- 如何相处：给稳定的日常仪式感（固定喂食、固定睡觉位置）；分离训练循序渐进
- 喂养指南：定时定量最重要，打乱节奏它会焦虑；可以用喂食仪式增强安全感
- 玩具与互动：陪伴型互动为主——抚摸、一起看窗外就好；不需要太复杂的玩具
- 情绪与心理：出门前低调，回家后先冷静再互动；给它你的旧衣服放窝里增加安全感

### ESTP — 飞檐走壁侠

**稀有度：** 7/10
**金句：** "那个柜子顶上我还没去过——等下，已经去过了。"
**标签：** 冒险王、行动派、爬酷高手
**气泡词：** 冒险、勇敢、活力、敏捷、自由、行动

**TA的内心独白：**
你问我为什么要跳到冰箱上面？因为它在那里啊。你问我为什么从冰箱上跳到书架上？因为我能啊。人生——不对，猫生就是要不断挑战自己。

**养护建议：**
- 家里必须有足够的垂直空间（猫爬架、跳台）
- 危险区域要做好防护（高窗、阳台）
- 充足的运动量是它情绪稳定的关键

- 每天同一时间叫你起床，误差不超过五分钟
- 固定在同一个位置睡觉，同一个位置吃饭
- 不会突然发疯或做出出格的事——行为高度可预测

**心理想的 vs 外表表现的：**
- 内心："六点半了，该起了。这是规矩。"→ 外表：准时跳上床，拍你的脸
- 内心："今天的流程跟昨天一样，很好"→ 外表：吃完饭去窗台趴着，跟昨天一模一样
- 内心："虽然日子重复，但每天有你就很好"→ 外表：每天晚上固定时间走到你身边趴下

**相处指南：**
- 如何相处：保持日常节奏稳定；改变任何事情都要循序渐进
- 喂养指南：固定时间固定地点固定分量——它的终极安全感来源
- 玩具与互动：规律性的互动——每天固定时间玩一次就好；不需要花哨
- 情绪与心理：突然的变化是它最大的压力源；搬家/换环境给足过渡时间
